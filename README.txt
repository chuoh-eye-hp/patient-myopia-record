# 患者用 近視記録入力ページ（GitHub Pages公開用）

## 使い方
このフォルダの中身をそのまま GitHub の新しいリポジトリへアップロードしてください。
その後、Settings → Pages → Deploy from a branch → main / root に設定すると公開できます。

## 中身
- index.html : 患者用ページ本体

## 注意
- 入力後は「JSON保存して提出」を押すと、提出用JSONを保存してGoogle Drive提出先を開きます
- Google Driveへの自動アップロードではなく、最後のアップロード操作は患者さん側で行います
